# 🎬 Movie Ratings Analysis

## 📌 Project Overview
This project analyzes movie data to explore factors that influence movie ratings. It uses real-world datasets to uncover patterns in ratings, genres, and popularity.

---

## 📊 Features
- Top 10 highest-rated movies visualization
- Genre-based rating analysis
- Correlation analysis between movie features
- Distribution of movie ratings

---

## 🗂️ Data Sources
- https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata
- https://www.kaggle.com/datasets/rounakbanik/the-movies-dataset

---

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

---

## 📈 Key Insights
- Some genres consistently receive higher ratings
- Popularity does not always mean higher rating
- Budget and revenue show moderate correlation

---

## 🚀 How to Run
1. Open the notebook in Jupyter
2. Run all cells
3. Explore the visualizations
