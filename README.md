# Movie Ratings Analysis

## Project Description
This project analyzes factors affecting movie ratings using real datasets.

## Data Sources
- https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata
- https://www.kaggle.com/datasets/rounakbanik/the-movies-dataset

## Methods Used
- Data cleaning
- Data analysis
- Visualization

## Results
The project explores how ratings are distributed and how they change over time.
